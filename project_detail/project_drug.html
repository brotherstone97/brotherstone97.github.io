딥러닝 기반 이미지 인식 기능을 활용한 약물 정보 제공 및 오남용 예방 모바일 앱

Repository

Data Crawler GitHub

Summary

약물 오남용으로 인한 부작용을 미연에 방지하고자 고안한 아이디어입니다.
사용자가 복용 중이거나 보관 중인 알약을 촬영해 앱에 업로드하면 이 약물에 대한 정보가 화면에 출력되도록 합니다. 또한 키워드 검색기능 역시 제공하여 사용자가 약물의 이름을 알고 있을 경우 해당 기능을 사용하여 약물 정보를 얻을 수 있습니다. 사용자에게 제공되는 약물정보는 효능효과, 부작용, 병용금기와 같이 가장 중요한 내용을 강조 및 최우선으로 표시하여 가독성을 최대한 높일 계획입니다.

Background

Detail

Review

2021년 한국정보산업연합회 주관 이브와ICT공모전 참가작입니다. 원래 계획대로라면 메뉴 추천 시스템 구현과 지도API를 이용한 가맹점 선택 기능 등을 추가했어야 했지만 마감기한 내 새로운걸 배우고 구현하는데 한계가 있어 기능을 축소할 수 밖에 없었던 점이 아쉬움으로 남습니다. 그럼에도 이번 프로젝트를 통해 REST API기반의 풀스택 개발을 경험해볼 수 있었습니다. 그리고 AWS의 여러가지 리소스를 사용해보며 Ubunt기반의 클라우드 환경과도 친숙해질 수 있었던 좋은 기회였습니다. 이번 대회에서는 2차예선까지 통과했지만 최종평가에서 수상하지 못하고 종료하게되었는데 다음 공모전에서는 완성도를 높혀서 입상하는 것이 목표입니다.
