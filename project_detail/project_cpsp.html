CPSP OCR 서버 및 이미지 업로드 End-point 개발(진행중)

Repository

OCR Server GitHub

Summary

현재 서비스 중인 친구의 프로젝트(사이퍼즈 서포터) 내 OCR기능 구현을 담당하게 되어 그 모델링과 이미지 수집 End-point를 개발하게 되었습니다.


Background

친구의 프로젝트 '사이퍼즈 서포터'는 온라인게임 '사이퍼즈'의 유저의 전적검색 기능을 제공하고 각종 통계 등 게임 플레이어들에게 유용한 정보를 제공하는 서비스입니다. 사이퍼즈라는 게임은 총 10명의 플레이어가 2팀을 이루어 5vs5 대전을 하는 방식으로, 게임 시작 전 전적검색 기능을 통해 상대 팀원의 전력을 분석하는 문화가 있습니다. 하지만 상대방의 닉네임을 일일이 입력해야하기 때문에 분석시간도 오래 걸리고 번거롭다는 문제점이 있습니다. 이를 극복하고자 게임 시작 전 화면에 모든 유저의 닉네임이 표시된다는 점을 이용하여 스크린샷 하나만으로 모든 유저의 전력을 분석할 수 있는 서비스를 생각하게 되었고, 스크린샷 속 유저 닉네임만을 추출하는 기능 개발을 담당하게 되었습니다.
